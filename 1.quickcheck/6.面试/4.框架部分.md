**操作DOM节点**：使用 ref

```html
<body>
    <div id="app">
        <div ref="hello">Hello World</div>
    </div>
</body>

<script>
let vm = new Vue({
    el: '#app',
    mounted () {
        console.log(this.$refs.hello.innerHTML) // Hello World
    }
})
</script>
```
