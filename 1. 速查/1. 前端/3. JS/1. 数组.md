# 2. 数组合并

### 方法一：concat方法（挂载Array原型链上）

```javascript

let a = [1,2,3];
let b = [4,5,6];

let c = a.concat(b);

console.log(c); // [1, 2, 3, 4, 5, 6]
console.log(a); // [1, 2, 3]  不改变本身

```

### 方法二：apply方法

```javascript
let a = [1,2,3];
let b = [4,5,6];

Array.prototype.push.apply(a, b);

console.log(a); // [1, 2, 3, 4, 5, 6] 改变原目标数组
console.log(b); // [4, 5, 6]

```
